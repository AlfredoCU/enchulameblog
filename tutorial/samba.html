    ¿Qué es Samba?

    Samba es un software que permite a tu ordenador con Ubuntu poder compartir archivos e impresoras con otras computadoras en una misma red local. Utiliza para ello un protocolo conocido como SMB/CIFS compatible con sistemas operativos UNIX o Linux , como Ubuntu, pero además con sistemas Windows (XP, NT, 98...), OS/2 o incluso DOS. También se puede conocer como LanManager o NetBIOS (Samba, Gua A.).

    Un servidor de archivos Samba permite compartir archivos en diferentes sistemas operativos a través de una red. Le permite acceder a sus archivos de escritorio desde una computadora portátil y compartir archivos con usuarios de Windows y macOS (Samba, Gua A.).
